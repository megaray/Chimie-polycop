## 1. RAPPEL DE THERMODYNAMIQUE

### 1.1 Entropie

**Définition** : L'entropie est une mesure du désordre d'un système.

**Points clés** :
- ✓ L'entropie du système ET des environs doit être considérée
- ✓ Les processus spontanés augmentent l'entropie totale de l'univers
- ✓ L'entropie est une fonction d'état

### 1.2 Énergie Libre de Gibbs

**Formule fondamentale** :

```
ΔG = ΔH − TΔS
```

**Signification des symboles** :

| Symbole | Signification | Unités |
|---------|---------------|--------|
| **ΔG** | Variation d'énergie libre de Gibbs | J, kJ |
| **ΔH** | Variation d'enthalpie | J, kJ |
| **T** | Température absolue | K |
| **ΔS** | Variation d'entropie | J·K⁻¹ |

**Critère de spontanéité** :
- ΔG < 0 : Réaction spontanée
- ΔG = 0 : Équilibre
- ΔG > 0 : Réaction non spontanée

### 1.3 Effet de la Température

**Quatre cas possibles** :

| ΔH | ΔS | Spontanéité | Conditions |
|----|----|--------------|-----------|
| < 0 | > 0 | Toujours spontané | À toute température |
| < 0 | < 0 | Spontané à basse T | T doit être faible |
| > 0 | > 0 | Spontané à haute T | T doit être élevée |
| > 0 | < 0 | Jamais spontané | À aucune température |

## 2. ÉNERGIE LIBRE ET VIE

### 2.1 Travail de Non-Expansion

**Exemple biologique** : Oxydation du glucose

```
C₆H₁₂O₆(s) + 6 O₂(g) → 6 CO₂(g) + 6 H₂O(l)
ΔG = −2879 kJ·mol⁻¹
```

**Application** :
- Construction de liens peptidiques : 17 kJ par mole
- 1 mole de glucose peut construire ~170 moles de liens peptidiques
- Plusieurs molécules de glucose nécessaires pour une protéine

### 2.2 ATP/ADP : La Monnaie Énergétique

**Cycle ATP/ADP** :

```
ATP → ADP + Pi        ΔG ≈ −30 kJ·mol⁻¹ (hydrolyse)
ADP + Pi → ATP        ΔG ≈ +30 kJ·mol⁻¹ (synthèse)
```

**Principe** :
- L'hydrolyse de l'ATP fournit l'énergie pour les réactions non spontanées
- La combustion du glucose "recharge" l'ADP en ATP
- ~80 moles d'ADP peuvent être rechargées par mole de glucose

**Analogie de la poulie** :
- Poids lourd (réaction spontanée) → descend et fournit de l'énergie
- Poids léger (réaction non spontanée) → monte grâce à l'énergie fournie
- Le couplage permet aux réactions non spontanées de se produire

## 3. TABLEAU PÉRIODIQUE DES ÉLÉMENTS

### 3.1 Principe de Construction

**Ordre de remplissage des orbitales** :

```
1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s → 5f → 6d → 7p
```

**Règles** :
1. Principe d'Aufbau : remplissage par ordre d'énergie croissante
2. Principe d'exclusion de Pauli : max 2 électrons par orbitale
3. Règle de Hund : orbitales dégénérées remplies avec spins parallèles d'abord

### 3.2 Blocs du Tableau Périodique

**s-block** : Groupes 1-2
- Métaux alcalins et alcalino-terreux
- Très réactifs
- Configuration : ns¹ ou ns²

**p-block** : Groupes 13-18
- Métaux, métalloïdes et non-métaux
- Configuration : ns²np¹⁻⁶

**d-block** : Groupes 3-12
- Métaux de transition
- Configuration : (n-1)d¹⁻¹⁰ns¹⁻²

**f-block** : Lanthanides et actinides
- Métaux de transition internes
- Configuration : (n-2)f¹⁻¹⁴

## 4. TENDANCES PÉRIODIQUES DES PROPRIÉTÉS ATOMIQUES

### 4.1 Charge Nucléaire Effective

**Concept** :
- Les électrons de valence subissent une charge nucléaire réduite
- Les électrons de cœur "écrantent" la charge nucléaire
- Zeff augmente de gauche à droite dans une période
- Zeff diminue en descendant un groupe

### 4.2 Rayon Atomique

**Tendances** :
- ↗ **Augmente en descendant un groupe** (plus de couches électroniques)
- ← **Diminue de gauche à droite** (Zeff augmente)

**Rayon ionique** :
- Cations : plus petits que l'atome neutre
- Anions : plus grands que l'atome neutre

### 4.3 Contraction Lanthanide

**Définition** : Diminution progressive des rayons atomiques des lanthanides due au remplissage des orbitales 4f.

**Conséquences** :
- Les éléments des Périodes 5 et 6 ont des rayons similaires
- Densités plus élevées pour les éléments de Période 6
- Faible réactivité de l'or et de l'argent

**Exemple** :
- Zr (Période 5) : rayon = 160 pm
- Hf (Période 6) : rayon = 156 pm (similaire malgré une période de différence)

### 4.4 Énergie de Première Ionisation

**Définition** : Énergie nécessaire pour arracher un électron d'un atome gazeux.

**Tendances** :
- ↗ **Augmente de gauche à droite** (Zeff augmente)
- ↓ **Diminue en descendant un groupe** (électrons plus éloignés du noyau)

**Formule** :
```
M(g) → M⁺(g) + e⁻        I₁ > 0
```

### 4.5 Affinité Électronique

**Définition** : Énergie libérée quand un électron est ajouté à un atome gazeux.

**Tendances** :
- Tendances moins régulières que pour l'énergie d'ionisation
- Affinités les plus élevées : haut à droite du tableau périodique
- Halogènes : affinités électroniques très élevées

### 4.6 Électronégativité

**Définition** : Tendance d'un atome à attirer les électrons de liaison.

**Échelle de Pauling** :
- F : 4.0 (la plus élevée)
- Cs : 0.7 (la plus faible)

**Tendances** :
- ↗ **Augmente de gauche à droite**
- ↓ **Diminue en descendant un groupe**

### 4.7 Polarisabilité

**Définition** : Facilité avec laquelle le nuage électronique d'un atome peut être déformé.

**Tendances** :
- ← **Diminue de gauche à droite** (électrons plus liés)
- ↓ **Augmente en descendant un groupe** (nuages électroniques plus diffus)

**Relation** :
- Atomes à haute polarisabilité : gros, riches en électrons
- Atomes à haute capacité polarisante : petits, fortement chargés

## 5. TENDANCES DANS LES PROPRIÉTÉS CHIMIQUES

### 5.1 Hydrures

**Trois types d'hydrures** :

#### A. Hydrures Salinsor Salt-like Hydrides)

**Caractéristiques** :
- Formés par les métaux des Groupes 1 et 2 (sauf Be)
- Composés ioniques M⁺H⁻
- Solides cristallins

**Préparation** :
```
2 K(s) + H₂(g) →[Δ] 2 KH(s)
Ca(s) + H₂(g) →[Δ] CaH₂(s)
```

#### B. Hydrures Métalliques

**Caractéristiques** :
- Formés par certains métaux du bloc d
- Solides noirs, poudreux, conducteurs
- Formule non stœchiométrique

**Applications** :
- Stockage et transport de l'hydrogène
- Libèrent H₂ par chauffage ou traitement acide

**Exemple** :
```
2 CuH →[Δ] x Cu + (2−x)CuH + ½x H₂(g)
CuH + HCl → CuCl + H₂(g)
```

#### C. Hydrures Moléculaires

**Caractéristiques** :
- Formés par les non-métaux
- Composés covalents
- Souvent volatils

**Exemples** :
- Gaz : NH₃, HF, HCl, HBr, HI, CH₄
- Liquides : H₂O, hydrocarbures (octane, benzène)

**Relation formule-groupe** :
```
Groupe 14 (IV) : CH₄
Groupe 15 (V) : NH₃
Groupe 16 (VI) : H₂O
Groupe 17 (VII) : HF
```

### 5.2 Oxydes

**Classification selon la position dans le tableau périodique** :

#### A. Oxydes Basiques

**Caractéristiques** :
- Formés par les métaux à faible énergie d'ionisation
- Ioniques, souvent solubles
- Réagissent avec l'eau pour former des bases

**Exemples** :
```
Na₂O + H₂O → 2 NaOH
CaO + H₂O → Ca(OH)₂
```

#### B. Oxydes Amphotères

**Caractéristiques** :
- Formés par Be, Al, et les métalloïdes
- Ne réagissent pas avec l'eau
- Solubles en milieu acide ET basique

**Exemples** :
```
Al₂O₃ + 6 HCl → 2 AlCl₃ + 3 H₂O
Al₂O₃ + 2 NaOH → 2 NaAlO₂ + H₂O
```

#### C. Oxydes Acides (Anhydrides)

**Caractéristiques** :
- Formés par les non-métaux
- Moléculaires, souvent gazeux
- Forment des acides dans l'eau

**Exemples** :
```
CO₂ + H₂O ⇌ H₂CO₃ ⇌ H⁺ + HCO₃⁻
SO₃ + H₂O → H₂SO₄
N₂O₅ + H₂O → 2 HNO₃
```

**Anhydrides formels** :
- CO : anhydride formel de HCOOH (ne réagit pas avec H₂O froid)

### 5.3 États d'Oxydation

**Tendances pour les éléments du bloc d** :

**Règles générales** :
1. Perte des électrons s en premier
2. Puis perte variable d'électrons d
3. Plusieurs états d'oxydation possibles

**Distribution** :
- Éléments au centre de chaque rangée : gamme la plus large
- Éléments des Périodes 5 et 6 : états d'oxydation plus élevés

**Relation avec le comportement acido-basique** :

| Oxyde | État d'oxydation | Comportement |
|-------|------------------|--------------|
| CrO | +2 | Basique |
| Cr₂O₃ | +3 | Amphotère |
| CrO₃ | +6 | Acide |

## 6. CLASSIFICATION ÉLECTRIQUE DES SOLIDES

### 6.1 Types de Conduction

#### A. Conduction Électronique

**Caractéristiques** :
- Charge transportée par les électrons
- Type de conduction dans les métaux et le graphite
- Résistance augmente avec la température

#### B. Conduction Ionique

**Caractéristiques** :
- Charge transportée par les ions
- Sels fondus ou solutions d'électrolytes
- Électrolytes solides : ions peuvent se déplacer dans le réseau

**Applications** :
- Batteries lithium-ion (électrolytes solides)

### 6.2 Classification selon la Résistance

**Trois catégories** :

#### A. Conducteurs Métalliques

**Propriétés** :
- Résistance augmente avec la température
- Électrons de conduction mobiles
- Conductivité élevée

#### B. Semiconducteurs

**Propriétés** :
- Résistance diminue avec la température
- Bande interdite petite
- Conductivité intermédiaire

#### C. Supraconducteurs

**Propriétés** :
- Résistance nulle en dessous de Tc
- Formation de paires de Cooper
- Tc généralement très basse

#### D. Isolants

**Propriétés** :
- Résistance très élevée
- Bande interdite large
- Pas de conduction électrique

## 7. THÉORIE DES ORBITALES MOLÉCULAIRES ET CONDUCTION

### 7.1 Formation des Bandes

**Principe** :
- N orbitales atomiques → N orbitales moléculaires
- Dans un solide, N ≈ 10²³ (énorme)
- Les OM sont si proches qu'elles forment une bande quasi-continue

**Structure en bandes** :

```
Énergie ↑
         |______________|  Bande de conduction (vide ou partiellement remplie)
         |              |
         |   Band gap   |  Bande interdite
         |______________|
         |//////////////|  Bande de valence (remplie ou partiellement remplie)
```

### 7.2 Bande de Valence

**Définition** : Bande d'orbitales moléculaires remplie par les électrons de valence.

**Caractéristiques** :
- Électrons liants
- Stabilise la structure
- Généralement remplie dans les isolants

### 7.3 Bande de Conduction

**Définition** : Bande d'orbitales moléculaires vide ou partiellement remplie.

**Caractéristiques** :
- Électrons mobiles
- Permet la conductivité
- Partiellement remplie dans les métaux

### 7.4 Bande Interdite (Band Gap)

**Définition** : Écart énergétique entre la bande de valence et la bande de conduction.

**Classification selon Eg** :

| Matériau | Eg | Comportement |
|----------|-----|--------------|
| **Conducteur** | 0 ou chevauchement | Conduction facile |
| **Semiconducteur** | 0.1 - 4 eV | Conduction thermiquement activée |
| **Isolant** | > 4 eV | Pas de conduction |

**Exemples** :
- Si : Eg = 1.1 eV (semiconducteur)
- GaAs : Eg = 1.4 eV (semiconducteur)
- Diamond : Eg = 5.5 eV (isolant)
- SiO₂ : Eg = 9 eV (isolant)

## 8. SEMICONDUCTEURS

### 8.1 Semiconducteurs Intrinsèques

**Définition** : Semiconducteur pur où la conductivité provient uniquement de l'excitation thermique.

**Mécanisme** :
```
Électron excité thermiquement :
Bande de valence → Bande de conduction
```

**Résultat** :
- Électron mobile dans la bande de conduction
- "Trou" (hole) mobile dans la bande de valence
- Conductivité augmente avec T

### 8.2 Dopage : Semiconducteurs Extrinsèques

**Principe** : Addition d'impuretés pour modifier les propriétés électriques.

#### A. Semiconducteur de type n

**Dopage** : Élément du Groupe 15 (As, P, Sb) dans Si ou Ge

**Mécanisme** :
- Atome dopant apporte un électron supplémentaire
- Électron faiblement lié, facilement excité vers la bande de conduction

**Diagramme de bandes** :
```
         |______________|  Bande de conduction
         |  • • • •     |  ← Électrons supplémentaires
         |--------------|  ← Niveau donneur (juste sous la BC)
         |              |
         |   Band gap   |
         |______________|
         |//////////////|  Bande de valence (remplie)
```

**Exemple** :
```
Si dopé avec As :
- Si : 4 électrons de valence
- As : 5 électrons de valence
- 1 électron "en trop" → facilement excité
```

#### B. Semiconducteur de type p

**Dopage** : Élément du Groupe 13 (B, Al, Ga, In) dans Si ou Ge

**Mécanisme** :
- Atome dopant crée un "trou" (manque un électron)
- Trou accepte facilement un électron

**Diagramme de bandes** :
```
         |______________|  Bande de conduction
         |              |
         |   Band gap   |
         |--------------|  ← Niveau accepteur (juste au-dessus de la BV)
         |      ○ ○     |  ← Trous
         |//////////////|  Bande de valence
```

**Exemple** :
```
Si dopé avec In :
- Si : 4 électrons de valence
- In : 3 électrons de valence
- 1 électron "manquant" → crée un trou
```

### 8.3 Jonction p-n

**Structure** : Interface entre un semiconducteur de type p et un de type n.

**Propriété fondamentale** : Le courant ne peut circuler que dans un sens.

#### A. Polarisation Inverse (Reverse Bias)

**Configuration** :
- Type p connecté au pôle négatif
- Type n connecté au pôle positif

**Résultat** :
- Les trous migrent vers le pôle négatif (côté p)
- Les électrons migrent vers le pôle positif (côté n)
- La jonction s'élargit
- **Pas de courant**

#### B. Polarisation Directe (Forward Bias)

**Configuration** :
- Type p connecté au pôle positif
- Type n connecté au pôle négatif

**Résultat** :
- Les trous sont repoussés vers la jonction
- Les électrons sont repoussés vers la jonction
- Recombinaison électron-trou à la jonction
- **Courant circule**

### 8.4 Applications des Jonctions p-n

#### A. Diodes

**Fonction** : Rectification du courant (passage dans un seul sens)

#### B. LED (Light-Emitting Diode)

**Principe** :
- Polarisation directe → recombinaison électron-trou
- Énergie libérée sous forme de photon
- Eg détermine la couleur de la lumière

**Relation énergie-couleur** :
```
Eg = hν = hc/λ
```

**Exemples** :
- GaN : Eg = 3.4 eV → UV/bleu
- GaAs : Eg = 1.4 eV → infrarouge
- GaP : Eg = 2.3 eV → vert

#### C. Cellules Solaires (Photovoltaïques)

**Principe** :
- Photon absorbé → création paire électron-trou
- Champ électrique à la jonction sépare les charges
- Courant électrique généré

## 9. SUPRACONDUCTEURS

### 9.1 Supraconductivité

**Définition** : Perte totale de résistance électrique en dessous d'une température critique Tc.

**Historique** :
- 1911 : Découverte (Hg, Tc = 4 K)
- 1988 : Tc = 125 K
- 2015 : Tc = 138 K
- 2020 : Tc = 287 K (mais à très haute pression)

### 9.2 Paires de Cooper

**Mécanisme** :
1. Un électron distord le réseau cationique
2. Distortion attire un deuxième électron
3. Formation d'une paire d'électrons faiblement liée
4. À basse T, vibrations du réseau réduites
5. Paires de Cooper ne sont pas séparées par les vibrations

**Propriétés** :
- Liaison très faible entre les deux électrons
- Stable uniquement à basse température
- Permet un transport sans résistance

### 9.3 Applications

**Avantages potentiels** :
- Transport d'électricité sans perte
- Aimants puissants (IRM, trains maglev)
- Dispositifs électroniques ultra-rapides

**Défis** :
- Températures critiques encore trop basses
- Matériaux fragiles
- Coût de refroidissement élevé

**Exemple pratique** :
- 25 kg de fil supraconducteur = 1800 kg de fil de cuivre
- Rapport de masse : 72:1

## CONSEILS POUR LA RÉSOLUTION DE PROBLÈMES

### ✓ Liste de Vérification

1. **Pour la thermodynamique** :
   - Identifier les signes de ΔH et ΔS
   - Déterminer la plage de températures de spontanéité
   - Vérifier les unités (J vs kJ, T en K)

2. **Pour les tendances périodiques** :
   - Repérer la position dans le tableau périodique
   - Considérer Zeff et le nombre de couches
   - Attention aux exceptions (contraction lanthanide)

3. **Pour les semiconducteurs** :
   - Identifier le type de dopage (n ou p)
   - Comprendre la direction du courant
   - Vérifier la polarisation (directe ou inverse)

### ⚠️ Erreurs Courantes

- ❌ Confondre ΔG et ΔH pour le critère de spontanéité
- ❌ Oublier que les tendances s'inversent pour les rayons ioniques
- ❌ Mélanger les porteurs de charge dans les semiconducteurs n et p
- ❌ Ignorer la contraction lanthanide pour les éléments de Période 6
